# Build should not exclude commons-logging anymore

## 内容

As of Spring Framework 7, `commons-logging:commons-logging` is an API dependency of `spring-core` that supersedes `spring-jcl`.

We therefore no longer need to exclude commons logging but we still have a few places where we do that.
