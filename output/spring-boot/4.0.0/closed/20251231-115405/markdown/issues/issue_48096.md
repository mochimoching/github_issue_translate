# Custom XML converters should override defaults in HttpMessageConverters

## 内容

Currently, a `JacksonXmlHttpMessageConverter` bean provided by the application will be configured as a custom converter, ahead of the default ones. It should be instead configured using the `withXmlConverter` builder method.
