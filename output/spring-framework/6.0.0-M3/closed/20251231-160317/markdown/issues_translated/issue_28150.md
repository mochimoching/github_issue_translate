## 内容

https://github.com/spring-projects/spring-framework/issues/28065 と https://github.com/spring-projects/spring-framework/issues/28088 に基づき、`GenericApplicationContext`を事前に処理するためのエントリーポイントを作成できるようになりました。

ジェネレーターは最初にAOT処理のためにコンテキストを更新し、次に貢献されたBean定義に適用可能な関連するコントリビューションを特定します。その後、コードとランタイムヒントが貢献されるようにそれらを呼び出します。
