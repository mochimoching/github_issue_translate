## 内容

コントリビューションには、命名や特権パッケージへのアクセスを気にせずにコードを貢献できる方法が必要です。そのような型にアクセスしてメソッドを貢献する方法を提供する、ある種の生成コンテキストが、その目標に向けた最初のステップです。
