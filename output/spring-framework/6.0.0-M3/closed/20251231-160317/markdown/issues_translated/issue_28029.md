## 内容

`BeanDefinitionValueResolver`は現在パッケージプライベートですが、AOTエンジンがこれを直接呼び出してコンストラクタ引数などを解決できるようにすることができます。

また、デフォルトの`TypeConverter`をデフォルトで提供できるようにすることで、現在の使用法を改善することもできます。現在のパターンは、Bean Factoryの保護されたメソッドを使用してそれを初期化しています。
