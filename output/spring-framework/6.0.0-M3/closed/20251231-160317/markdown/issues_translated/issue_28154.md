# 曖昧性チェックがコンストラクタにのみ適用される

## 内容

Beanインスタンスを作成するためにコンストラクタを呼び出す必要がある場合、異なるパラメータ型を持つ複数のバリエーションが存在し、型が宣言されていないため、`new MyBean(attributes.get(0)`への呼び出しが曖昧になる危険性があります。

このケースに対する明示的なチェックがあり、`InjectedElementAttributes`の明示的な型ベースのメソッドが使用されます。残念ながら、同様のシグネチャを持つ複数のメソッドがある場合、同じ問題を抱えるファクトリメソッドには適用されていません。
