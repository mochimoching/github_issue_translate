## 内容

ビルド時に`ApplicationContext`を事前処理できるようにするには、Beanインスタンスを作成する準備が整う時点まで「更新」する方法が必要です。これには以下が含まれます:

* `BeanFactory`の準備
* `BeanDefinitionRegistryPostProcessor`実装の呼び出し
* `MergedBeanDefinitionPostProcessor`実装の呼び出し(これらは(マージされた)`BeanDefinition`レベルで動作する`BeanPostProcessor`の特別な拡張です)
