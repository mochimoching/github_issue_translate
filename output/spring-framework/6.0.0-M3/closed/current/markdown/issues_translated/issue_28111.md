## 内容

コンテキストがAOTエンジンによって事前処理される場合、実行時には設定クラスの解析が行われなくなり、`ImportAware`設定クラスとそれをインポートしたクラスとの間のリンクを保持する必要があります。

これは、マッピングを計算し、コールバックを尊重するポストプロセッサーを提供することで実現できます。
