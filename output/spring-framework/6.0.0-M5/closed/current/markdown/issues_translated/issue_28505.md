## 内容

現在、`HttpServiceProxyFactory`はプログラマティックな使用に適していますが、`ConversionService`や#28492の時点での`EmbeddedValueResolver`など、多数の依存関係を注入する必要があるため、Beanとして宣言される可能性が高いです。Bootアプリケーションでは、`WebClient.Builder`も注入することになります。
