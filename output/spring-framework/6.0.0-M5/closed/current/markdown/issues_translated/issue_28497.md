## 内容

コアフレームワークは`@AliasFor`の使用を許可しており、これにより裏でJDKプロキシが作成されます。アノテーションの属性にリフレクションヒントを登録するだけでは、ネイティブイメージで実行時にこれを使用するには不十分です。

登録を容易にするために、このロジックを処理するユーティリティを追加します。
