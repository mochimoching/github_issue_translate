# BeanRegistrationAotProcessorビーンのフィルタリングはオプションにすべき

## 内容

現在、`BeanDefinitionMethodGeneratorFactory.isImplicitlyExcluded`は常に`BeanRegistrationAotProcessor`を実装するビーンをフィルタリングします。Spring Bootでは、`BeanRegistrationAotProcessor`を実装するが、フィルタリングされないビーンが必要になると考えています。

## コメント

### コメント 1 by philwebb

**作成日**: 2022-05-27

これには`BeanRegistrationExcludeFilter`を信号として使用できると思います。`BeanRegistrationExcludeFilter`を実装していないAOTプロセッサービーンは暗黙的にフィルタリングできます。`BeanRegistrationExcludeFilter`を実装している場合は、それ自身のフィルタリングを処理しているという信号と見なします。
