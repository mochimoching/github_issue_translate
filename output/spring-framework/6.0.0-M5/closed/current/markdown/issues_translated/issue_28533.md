## 内容

単一の場所から例外を処理する良い方法がありません。

アプリケーションは`WebClientException`を別の例外に変換したり、特定の例外を処理したりしたい場合があります。`WebClient`はステータスハンドラーを提供していますが、これらはリクエストごとに設定され、ビルダーを通じて公開されていません。
