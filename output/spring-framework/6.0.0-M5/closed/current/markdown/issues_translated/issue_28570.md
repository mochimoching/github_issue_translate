# BeanDefinitionPropertiesCodeGeneratorは単一の推測されたinit/destroyメソッドをフィルタリングすべき

## 内容

`BeanDefinitionPropertiesCodeGenerator`は現在、単一の`AbstractBeanDefinition.INFER_METHOD`メソッドのみが指定されている場合、パラメータなしの`setInitMethodNames()/setDestroyMethodNames()`メソッドを生成します。これは無害ですが、見栄えが良くないため、ステートメントを追加しないようにすべきです。
