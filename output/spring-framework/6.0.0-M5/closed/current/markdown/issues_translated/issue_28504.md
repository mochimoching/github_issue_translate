## 内容

`@HttpExchange`は現在、共通属性を継承するためにクラスレベルでのみ許可されていますが、メソッドレベルでは、HTTPメソッド固有のショートカットアノテーションのいずれかを選択する必要があります。カスタムHTTPメソッドや、専用のショートカットを必要としないOPTIONSのようなあまり一般的でないメソッドのために、メソッドレベルで許可できます。
