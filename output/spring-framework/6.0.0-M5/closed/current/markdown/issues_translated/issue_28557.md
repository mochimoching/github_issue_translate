# BeanRegistrationAotContributionsがBeanRegistrationCodeFragmentsのカスタマイズを提供できるようにする

## 内容

現在、`BeanRegistrationCodeFragments`をカスタマイズできるのは`BeanRegistrationCodeFragmentsCustomizer`のみですが、親/子コンテキストサポートのために、`BeanRegistrationAotContribution`から直接カスタマイズを適用できるようにしたいと考えています。

このような変更により、既存の`BeanRegistrationAotProcessor`インターフェースを使用して同じ機能を実現できるため、`BeanRegistrationCodeFragmentsCustomizer`を完全に削除できるはずです。
