## 内容

`native-image.properties`は、`,`で区切られた複数のパッケージを含む単一の`--initialize-at-build-time`パラメータの指定をサポートしています。この課題は、この機能を使用することを意図しています。理由は以下の通りです：

 - これは通常、複数のパッケージを指定する推奨方法です。
 - これは生成された設定と一貫性があります。
 - `native-image`コマンドに渡されたときに、モジュールごとに`--initialize-at-build-time`をグループ化できます。
