## 内容

[#19628](https://github.com/spring-projects/spring-framework/issues/19628)のフォローアップとして、RxJava 1.xは既にRxJava 2.xだけでなくRxJava 3.xにも置き換えられているため、Springの`ReactiveAdapterRegistry`からRxJava 1.xサポートを最終的に削除します。

後の注記により、RxJava 2は2021年2月にEOLを迎えているため、RxJava 2.xのサポートも即座に削除します。
