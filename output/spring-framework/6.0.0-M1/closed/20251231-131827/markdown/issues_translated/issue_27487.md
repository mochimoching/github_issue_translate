## 内容

**影響バージョン:** 6.0 snapshots

`AbstractJpaVendorAdapter`はJPA 2.1を参照していますが、現在はJPA 3.0が必要です。同様に、`HibernateJpaVendorAdapter`はHibernate 5.2、5.3、5.4のサポートを説明していますが、現在は最低でも5.5が必要と考えられます。5.5が`-jakarta`バリアントのモジュールを持ち、JPA 3.0をサポートする最も早いバージョンだからです。
