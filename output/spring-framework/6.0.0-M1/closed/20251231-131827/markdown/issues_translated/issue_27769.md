# StringUtils::trimWhitespaceを非推奨化

## 内容

PR [#27703](https://github.com/spring-projects/spring-framework/pull/27703)により、`StringUtils::trimWhitespace`が些細なものになりました。このメソッドは現在`String::strip`に委譲しているためです。`trimWhitespace`の唯一の意義は`null`を受け入れることです。`String::strip`を推奨して`trimWhitespace`を非推奨化するべきです。

同様に、`String::stripLeading`を推奨して`StringUtils::trimLeadingWhitespace`を、`String::stripTrailing`を推奨して`StringUtils::trimTrailingWhitespace`を非推奨化するべきです。
