## 内容

[JAMon](https://github.com/stevensouza/jamonapi)プロジェクトはもうあまりアクティブではないようです。最後の更新は(ほぼ)2年前で、Spring 6がリリースされる頃には3年になります。新しいJavaバージョンやその後にリリースされたAPIバージョンを完全にサポートしているかどうかさえわかりません。

現在では、JavaMelody、Micrometer、Sentry.ioなど、より洗練されたソリューションがあります。このネイティブサポートを削除することを検討する価値があるかもしれません(インターセプタをJAMonプロジェクトに寄贈することも考えられます)。
