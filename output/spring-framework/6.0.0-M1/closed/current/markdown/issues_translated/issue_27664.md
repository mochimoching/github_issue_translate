## 内容

[#27646](https://github.com/spring-projects/spring-framework/issues/27646)に関連して、`EncodedResource`は文字列エンコーディング名ではなく`StandardCharsets`引数で一貫して使用されるべきです。また、JDK 10+では`Charset`引数を持つ`URLEncoder`/`URLDecoder`のバリアントが利用可能になっています。
