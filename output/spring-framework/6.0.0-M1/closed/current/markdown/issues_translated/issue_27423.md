## 内容

Webサポートにおけるいくつかの統合オプションは2005年代に遡るもので、近年メンテナンスが行われておらず、Jakarta Servlet(`jakarta.servlet`パッケージ名前空間)へのアップグレードも得られていないようです。Jakarta EE 9+への改訂の一環として、CommonsMultipartResolver、FreeMarkerServletスタイルのJSP taglibサポート、Tilesビューのサポートを削除します。
