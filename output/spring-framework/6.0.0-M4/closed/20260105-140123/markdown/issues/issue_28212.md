## 内容

As discovered by @snicoll, serialiation of `queriedMethods` is broken when there is no `methods`.
```json
[
  {
    "name": "com.example.nativex.sample.basic.SampleConfiguration"
    "queriedMethods": [
      {
        "name": "testBean",
        "parameterTypes": []
      }
    ]
  }
]
```
